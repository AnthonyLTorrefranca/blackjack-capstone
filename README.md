# Blackjack Capstone

A command-line Blackjack game built with Python as part of Angela Yu's 100 Days of Code course.

## Project Goal
Reinforce core Python concepts through a fun, interactive card game.

## Concepts Practiced
- Functions and scope
- Lists and mutation
- Control flow (if/else, loops)
- Randomization (random module)

## How to Run
python main.py

## Game Rules
- Get as close to 21 without going over
- Ace counts as 11, but switches to 1 if total exceeds 21
- Dealer must draw until they reach 17 or more
- A hand of exactly 21 with 2 cards is a Blackjack
